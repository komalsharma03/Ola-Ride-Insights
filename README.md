# OLA Ride Insights

## Project Overview

This project analyzes OLA ride booking data using:

- Python
- SQL
- Power BI
- Streamlit

The dashboard provides:

- Ride trends
- Revenue analysis
- Ratings analysis
- Cancellation analysis
- Customer insights

- Power BI
