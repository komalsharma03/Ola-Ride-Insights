# OLA Ride Insights Dashboard

## Project Overview

OLA Ride Insights is a complete Data Analytics project built using:

- Python
- MySQL
- Power BI
- Streamlit

The project analyzes ride-sharing booking data to generate business insights related to:
- ride demand
- revenue trends
- peak hours
- customer behavior
- payment methods
- ratings analysis

This project demonstrates the end-to-end analytics workflow including:
- data cleaning
- SQL database management
- dashboard development
- web app deployment
